=>Git vs Github/Gitlab
Git - git is a tool to manage github/gitlab repositories (git can use without github/gitlab)
	- store versions in project history

Github/Gitlab - place where the source code exists (remotely and manage by git)
			  - online service that hosts our projects
			  - share our codes to other developers
			  - download / collaborate
