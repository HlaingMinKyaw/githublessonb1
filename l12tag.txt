=>Git Tag 
(i)lightweight Tags - A Name Only (get from commit)
(ii)Annotated Tags - Allows yo to adda message explaining the tag 

(i)lightweight Tags
git tag tagname(by editor)

(ii)Annotated Tags 
git tag -a tagname  -m "message" 
(or)
git tag -a tagname 

=> Delete 
git tag --delete tagname 
or 
git tag -d tagname
